## 实验五 diffie-hellman密钥协商算法

### 算法过程
1.生成一个素数p和p的原根g
2.Alice和Bob分别生成一个随机数作为自己的私钥X
3.Alice和Bob使用自己的私钥生成公钥Y: **Y=g ** X % p**
4.Alice和Bob使用对方的公钥Y和自己的私钥X计算出最终的密钥K：**K=Y ** X %p**
